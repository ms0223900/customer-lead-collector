### US-11：管理者可以將展示版文案套用到銷售頁預覽

**作為** 管理者  
**我想要** 將展示版產生的文案套用到銷售頁預覽  
**以便** 理解文案如何影響 Landing Page 畫面

**輸入格式**：
- US-10 產出的 Mock 文案物件（Hero 標題、副標、痛點、承諾、CTA、FAQ）
- 管理者點擊「套用到預覽」

**輸出格式**：
- Landing Page 預覽區更新對應文案區塊
- 僅影響前端預覽或 mock 狀態，不正式發布至對外 `/`
- 頁面標示「預覽效果」；重新整理可回到預設文案

**驗收條件**：
- [ ] 文案產生結果旁有「套用到預覽」按鈕
- [ ] 點擊後，預覽區更新 Hero 標題、副標、CTA 或 FAQ 等對應區塊
- [ ] 更新只影響預覽狀態，不修改對外正式 Landing Page 持久資料
- [ ] 頁面清楚標示這是預覽效果
- [ ] 重新整理後可回到預設狀態（或提供「還原預設」）

**依賴關係**：
- US-10：Mock 文案產生器
- US-01：Landing Page 區塊結構

**優先級**：P2（Could Have；影片時間不足可略過）  
**相關功能**：銷售頁文案套用預覽（spec §4 Could Have）

**備註**：
- US-10 已足夠展示「AI 功能先做展示版」思路；本 US 為加分項。
