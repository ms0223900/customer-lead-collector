### US-07：管理者可以查看名單列表

**作為** 管理者  
**我想要** 在後台看到所有等候名單  
**以便** 知道目前有哪些潛在學員

**輸入格式**：
- 管理者進入 `/admin`（或 spec 定義之後台路由）
- 後端自 Supabase `leads` 讀取名單列表（server 端，使用適當 credentials）

**輸出格式**：
- 後台名單列表表格或清單，每筆顯示：姓名、Email、身分、狀態、建立時間
- 依 `created_at` 降序排列（最新在前）
- 無資料時顯示空狀態（繁中提示）

**驗收條件**：
- [ ] 後台顯示名單列表
- [ ] 每筆名單顯示姓名、Email、身分、狀態、建立時間
- [ ] 最新名單排在前面
- [ ] 沒有資料時顯示空狀態
- [ ] 列表資料來自 `leads` 資料表，非前端假資料
- [ ] 資料庫讀取失敗時，UI 顯示可讀繁中錯誤訊息

**依賴關係**：
- US-05：`leads` 資料可寫入
- Sprint 0：後台路由骨架、Supabase 連線

**優先級**：P0  
**相關功能**：名單管理 Admin Panel（spec §6.3、§8 管理者後台）
