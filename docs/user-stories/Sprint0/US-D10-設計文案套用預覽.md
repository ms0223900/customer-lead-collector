### US-D10：設計文案套用預覽

**作為** 負責 MVP 圖面設計的設計師  
**我想要** 完成 Mock 文案套用至銷售頁預覽的設計稿  
**以便** 若實作 Sprint 5 可選項 US-11 時，預覽與還原行為有明確對照

**輸入格式**：
- US-D09 AI 文案產生器展示版（Mock 結果區）
- US-D02 Landing Page 區塊結構（Hero、痛點、CTA、FAQ 等可替換區塊）

**輸出格式**：
- 設計稿含：
  - US-D09 結果區旁「套用到預覽」按鈕
  - **預覽區**：Landing 縮小版或並排預覽，對應區塊顯示 Mock 文案
  - 頂部「預覽效果」標籤
  - 「還原預設」按鈕或註解：重新整理可回到預設文案

**驗收條件**：
- [ ] 「套用到預覽」按鈕位置明確
- [ ] 預覽區可看出 Hero 標題／副標／CTA 或 FAQ 等已更新
- [ ] 「預覽效果」標籤醒目，與正式對外 `/` 區分
- [ ] 有還原預設的 UI 或行為註解
- [ ] 稿面標註：不修改對外正式 Landing 持久資料
- [ ] 稿面標註對應實作 US：US-11

**依賴關係**：
- US-D09：AI 文案產生器展示版
- US-D02：Landing Page 區塊結構

**優先級**：P2  
**相關功能**：銷售頁文案套用預覽（spec §4 Could Have）

**備註**：
- 影片時間不足可略過；US-D09 已足夠展示「AI 先做展示版」教學概念。
- 若僅實作 Sprint 1 + Sprint 2，**可略過**。
